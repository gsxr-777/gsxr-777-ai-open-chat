# GSXR-777 AI Open Chat - WordPress Plugin

## Обзор проекта

GSXR-777 AI Open Chat - это универсальный WordPress плагин для интеграции AI чат-бота на сайт с поддержкой множества провайдеров LLM моделей. Плагин создан на основе существующего проекта ai-chat-widget и адаптирован для экосистемы WordPress.

## Ключевые особенности

### 🤖 Универсальная поддержка AI
- **OpenAI** (GPT-4, GPT-3.5, GPT-4o-mini)
- **Anthropic Claude** (Claude 3 Sonnet, Haiku, Opus)
- **Google Gemini** (Gemini Pro, Flash)
- **GigaChat** (Sber)
- **YandexGPT** (Yandex)
- **Ollama** (локальные модели)
- **OpenRouter** (множественные провайдеры)
- **Любой OpenAI-совместимый API**

### 🎨 Полная кастомизация
- Настройка цветов, позиции, размеров
- Двуязычный интерфейс (русский/английский)
- Адаптивный дизайн для мобильных устройств
- Поддержка шорткодов для размещения

### 📚 База знаний
- Хранение Markdown-документов в базе WordPress
- Индексация опубликованных записей и страниц
- Разбиение источников на фрагменты и полнотекстовый поиск с fallback
- Передача модели только релевантных источников

### 🔒 Безопасность
- Аутентифицированное шифрование API-ключей AES-256-GCM
- Подписанные сервером идентификаторы сессий
- Атомарные лимиты запросов по IP и сессии
- Изоляция контента сайта от системных инструкций
- Валидация, санитизация и подготовленные SQL-запросы

### 📊 Аналитика
- Статистика сессий и сообщений
- Графики активности
- Отслеживание использования токенов
- Экспорт данных

## База данных

### Таблицы
1. **wp_gsxr777_sessions** - сессии чата
2. **wp_gsxr777_messages** - сообщения
3. **wp_gsxr777_security_log** - события безопасности
4. **wp_gsxr777_blocked_ips** - временные блокировки
5. **wp_gsxr777_rate_limits** - счетчики лимитов
6. **wp_gsxr777_knowledge_documents** - ручные документы
7. **wp_gsxr777_knowledge_chunks** - поисковый индекс mini-RAG

### Опции WordPress
- `gsxr_777_api_*` - настройки API
- `gsxr_777_widget_*` - настройки виджета
- `gsxr_777_rate_limit_*` - настройки безопасности

## Безопасность

### Защита от атак
- **XSS** - экранирование вывода
- **SQL Injection** - подготовленные запросы
- **CSRF** - nonce токены
- **Prompt Injection** - недоверенный контент отделён от системных инструкций
- **DoS** - ограничение частоты запросов

### Приватность
- **Шифрование** - API ключи в БД
- **Контроль контекста** - настройка передачи данных
- **Минимизация данных** - метаданные отключены по умолчанию
- **Срок хранения** - автоматическая очистка и удаление истории посетителем
- **Правовой статус** - плагин предоставляет настройки приватности, но не заменяет юридический аудит

## Установка и настройка

### Системные требования
- WordPress 5.0+
- PHP 7.4+
- MySQL 5.6+
- 128MB RAM (рекомендуется 256MB)

### Быстрый старт
1. Установить и активировать плагин
2. Настроить API в AI Chat > API Settings
3. Протестировать соединение
4. Кастомизировать виджет в Widget Settings
5. Добавить базу знаний (опционально)

### Проверки для разработчика

```bash
composer install
composer lint
composer security-check
composer phpcs
composer test
composer pot
```

Для `composer test` требуется стандартная тестовая среда WordPress и переменная
`WP_TESTS_DIR`. Полный `composer phpcs` также показывает оставшиеся форматные
нарушения старого кода; security-набор вынесен в отдельную обязательную проверку.

## Использование

### Автоматическое отображение
Виджет автоматически появляется на всех страницах сайта.

### Шорткоды
```php
// Базовое использование
[gsxr_777_chat]

// С параметрами
[gsxr_777_chat title="Поддержка" color="#ff6b35" width="420" height="640"]
```

### Хуки для разработчиков
```php
// Увеличить число найденных фрагментов.
add_filter('gsxr_777_rag_chunk_limit', function() {
    return 8;
});

// Доверять заголовкам прокси только от явно заданной сети.
add_filter('gsxr_777_trusted_proxy_ips', function() {
    return array('10.0.0.0/8');
});
```
