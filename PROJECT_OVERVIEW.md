# GSXR-777 AI Open Chat - WordPress Plugin

## Обзор проекта

GSXR-777 AI Open Chat - это универсальный WordPress плагин для интеграции AI чат-бота на сайт с поддержкой множества провайдеров LLM моделей. Плагин создан на основе существующего проекта ai-chat-widget и адаптирован для экосистемы WordPress.

## Ключевые особенности

- **OpenAI** (GPT-4, GPT-3.5, GPT-4o-mini)
- **Anthropic Claude** (Claude 3 Sonnet, Haiku, Opus)
- **Google Gemini** (Gemini Pro, Flash)
- **GigaChat** (Sber)
- **YandexGPT** (Yandex)
- **Ollama** (локальные модели)
- **OpenRouter** (множественные провайдеры)
- **Любой OpenAI-совместимый API**

## База данных

### Таблицы
1. **wp_gsxr777_sessions** - сессии чата
2. **wp_gsxr777_messages** - сообщения
3. **wp_gsxr777_security_log** - события безопасности

### Опции WordPress
- `gsxr_777_api_*` - настройки API
- `gsxr_777_widget_*` - настройки виджета

## Установка и настройка

### Системные требования
- WordPress 5.0+
- PHP 7.4+
- MySQL 5.6+
- 128MB RAM (рекомендуется 256MB)

### Быстрый старт
1. Установить и активировать плагин
2. Настроить API в AI Chat > API Settings
3. Протестировать соединение
4. Кастомизировать виджет в Widget Settings
5. Добавить базу знаний (опционально)

## Использование

### Автоматическое отображение
Виджет автоматически появляется на всех страницах сайта.

### Шорткоды
```php
// Базовое использование
[gsxr_777_chat]

// С параметрами
[gsxr_777_chat title="Поддержка" position="bottom-left" color="#ff6b35"]
```

### Хуки для разработчиков
```php
// Модификация системного промпта
add_filter('gsxr_777_system_prompt', function($prompt, $context) {
    return $prompt . "\n\nДополнительные инструкции.";
}, 10, 2);

// Настройка виджета
add_filter('gsxr_777_widget_config', function($config) {
    $config['customOption'] = 'value';
    return $config;
});
```
